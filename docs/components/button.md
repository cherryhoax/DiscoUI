# DiscoButton (`<disco-button>`)

A standard push button with the Disco UI visual style and tilt effect.

![DiscoButton](../../assets/components/button.webp)

## Usage

```html
<disco-button>Click Me</disco-button>
```

## Events

- Standard `click` event.

## Features

- **Tilt Effect**: The button executes a 3D tilt animation on press (simulating Windows Phone interaction).
