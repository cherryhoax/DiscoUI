# Password Box

The `disco-password-box` component provides a password input field with Metro visual style. It inherits from [Text Box](text-box.md) but enforces the password type.

## Usage

```html
<disco-password-box placeholder="Password"></disco-password-box>
```

## Attributes

| Attribute     | Description                             |
| :------------ | :-------------------------------------- |
| `value`       | The current value of the password input. |
| `placeholder` | Placeholder text to display when empty. |

## Properties

| Property      | Description                        |
| :------------ | :--------------------------------- |
| `value`       | Gets or sets the value.            |
| `placeholder` | Gets or sets the placeholder text. |

## Styling

Same as `disco-text-box`.
